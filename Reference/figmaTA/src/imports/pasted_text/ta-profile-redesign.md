Redesign this “TA Portal – Profile” page into a more Apple-inspired interface while preserving the existing information architecture, layout logic, and functionality.

The current page contains:
- A left sidebar navigation with “Profile”, “Jobs”, and “Status”
- A top header with a welcome message and logout action
- A main profile page title and supporting description
- A large profile information card
- A user avatar area with name, program, and “Edit Photo” button
- A two-column form layout with text inputs
- A large “Academic Background” textarea
- A primary “Save Changes” button

Please keep the structure recognizable, but redesign the full visual system to feel much closer to Apple’s design language: minimal, premium, refined, soft, spacious, and elegant.

OVERALL DESIGN GOAL
Transform this page from a standard institutional admin dashboard into a clean, high-end, Apple-style education platform interface. The final result should feel like a blend of:
- Apple website visual restraint
- iOS / iPadOS Settings elegance
- macOS panel clarity
- premium SaaS product polish

VISUAL STYLE DIRECTION
Use a minimalist and highly refined visual language with:
- generous whitespace
- soft rounded corners
- subtle depth
- clean typography hierarchy
- low-contrast surfaces
- restrained use of color
- modern, calm, premium UI tone

The interface should feel:
- clean
- light
- balanced
- intelligent
- soft but not playful
- premium but not flashy
- modern but not futuristic

Avoid anything that feels:
- generic enterprise dashboard
- outdated university admin system
- boxy or rigid
- crowded
- heavily bordered
- over-decorated
- saturated or loud
- cheap or template-like

COLOR SYSTEM
Use an Apple-like neutral palette:
- off-white, soft white, mist gray, light neutral backgrounds
- soft graphite or dark gray for headings and key text
- medium gray for secondary labels and descriptions
- very subtle border gray for inputs and dividers
- one restrained accent color, preferably a soft Apple-like blue, used sparingly for:
  - selected states
  - focus states
  - primary call-to-action
  - small interface highlights

Color should be minimal and intentional.
Do not use strong gradients, harsh contrast, or highly saturated blocks of color.
If gradients are used at all, they must be extremely subtle and premium.

SURFACE TREATMENT
Update all surfaces to feel softer and more modern:
- cards should use larger, unified corner radii
- inputs and buttons should be rounded and refined
- shadows should be very subtle and diffuse
- borders should be thin and low contrast
- surfaces may include light tonal separation instead of strong outlines
- use depth sparingly to suggest hierarchy without heaviness

The page should feel airy and layered, not flat and mechanical.

TYPOGRAPHY
Refine typography to feel more premium and Apple-like:
- strong but clean page title
- compact, elegant section hierarchy
- highly readable form labels
- softer secondary text
- balanced spacing between headings, labels, and controls

Typography should communicate clarity and calm.
Do not make the page feel overly bold or overly corporate.
Use visual hierarchy through size, weight, spacing, and contrast rather than decorative styling.

LAYOUT AND SPACING
Keep the same overall layout, but improve spacing significantly:
- increase breathing room around major sections
- align all content with more precision
- create more visual rhythm between title area, profile header, form groups, and action area
- make spacing more intentional and consistent
- improve horizontal and vertical padding throughout the page
- preserve usability while making the page feel less dense and more luxurious

The entire page should feel easier to scan and more comfortable to use.

LEFT SIDEBAR NAVIGATION
Redesign the sidebar to feel modern, soft, and premium:
- maintain the same navigation items and hierarchy
- simplify and elevate the visual treatment
- use more refined spacing and alignment
- selected state should be clearly visible but subtle
- selected item can use a soft tinted background, rounded pill/card treatment, or understated highlight
- icons should be cleaner, lighter, and more consistent
- reduce the feeling of a flat vertical menu and make it feel more like a polished system sidebar

The sidebar should resemble a premium product navigation system rather than a traditional admin menu.

TOP HEADER
Redesign the top header with more visual polish:
- keep the welcome text and logout action
